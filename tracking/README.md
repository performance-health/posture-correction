# Progress Tracking

Templates for monitoring compliance and progress.

## Templates

- `weekly-log-template.md` - Daily exercise tracking
- `monthly-review-template.md` - Monthly progress summary
- `progress-tracking-template.md` - Long-term milestone tracking

## Usage

### Weekly Tracking
```bash
cp weekly-log-template.md my-week-1.md
# Update daily as you complete exercises
```

### Monthly Review
```bash
cp monthly-review-template.md my-month-1.md
# Complete at end of each month
```

## Privacy

Files matching `my-*.md` and date patterns are gitignored.
