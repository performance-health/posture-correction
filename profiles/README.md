# Patient Profiles

Directory for managing patient/personal profiles.

## Files

- `profile-template.md` - Base template for creating new profiles
- `patient-*.md` - Individual patient files (gitignored for privacy)

## Usage

### Create New Profile

```bash
cp profile-template.md patient-001.md
# Edit patient-001.md with actual data
```

### Privacy Note

All files matching `patient-*.md` are automatically excluded from git commits to protect privacy.
