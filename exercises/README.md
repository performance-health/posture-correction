# Exercise Library

Categorized database of corrective exercises.

## Categories

### Mobility
Decompression, stretching, and mobility work
- Foam rolling
- Static stretches
- Dynamic mobility drills

### Strengthening
Strength and stability exercises
- Scapular stabilization
- Core strengthening
- Posterior chain development

### Integration
Functional movement patterns
- Compound movements
- Proprioception
- Sport/activity-specific

## Exercise File Format

Each exercise should include:
- Purpose/target muscles
- Equipment needed
- Step-by-step instructions
- Sets/reps/duration
- Progressions/regressions
- Common mistakes
- Safety notes
