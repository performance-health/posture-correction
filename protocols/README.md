# Rehabilitation Protocols

Evidence-based protocols organized by phase and severity.

## Available Protocols

- `full-intensive-protocol.md` - Complete 16+ week program (recommended)
- `phase-1-decompression.md` - Weeks 1-8: Mobility and decompression
- `phase-2-strengthening.md` - Weeks 8-16: Strength and stability
- `phase-3-integration.md` - Weeks 16+: Functional integration

## Protocol Selection

| Posture Score | Recommended Protocol |
|---------------|---------------------|
| 1-2 | Medical evaluation + Full Intensive |
| 3-4 | Full Intensive Protocol |
| 5-6 | Phase-based approach (1→2→3) |
| 7-8 | Targeted exercises only |

## Usage

1. Complete assessment first
2. Select protocol based on score
3. Follow progressions strictly
4. Track compliance and progress
